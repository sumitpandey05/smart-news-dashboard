# Smart News Intelligence Dashboard

A modern **React-based news analytics dashboard** that transforms the way users interact with news.  
Instead of just reading articles, users can explore trends, analyze sentiment, search topics, and manage personalized bookmarks in a clean and interactive interface.

This project combines **news exploration + lightweight analytics** to provide a smarter way to consume information.

---

## Features

### Dynamic News Feed
- Browse articles across multiple categories
- Smooth scrolling with load-more functionality
- Read time estimator for articles
- Category filtering (Tech, AI, Business, World, etc.)

### Smart Article View
- AI-style article summaries
- Sentiment indicators (positive / neutral / negative)
- Clean article reading interface

### Search & Discovery
- Search articles by keywords or topics
- Explore trending keywords
- Quickly filter results based on category

### Personalization
- Bookmark articles
- Track reading history
- Local storage based user data

### Analytics Dashboard
- Sentiment distribution visualization
- Category based article insights
- Trending keyword tracking

---

**Frontend**
